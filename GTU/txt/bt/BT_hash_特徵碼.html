教您該如何使用 BT info hash 特徵碼，下載 torrent 種子檔

使用時機，用 Google 搜索引擎，查找到某某資源之後，但找不到 .torrent 種子檔文件要在那裡下載...等各種情況，這時可以看一下網頁上，是否有提供 .torrent 種子檔文件之 info hash 哈希特徵碼；如果有的話，可以使用該頁面之磁力連結產生器，生成一串 magnet link 磁連網址

English Version https://zfly9.blogspot.com/2018/05/magnet-link.html


Magnet Link Generator 磁鏈產生器 make a magnet link from a torrent info hash
請於下框內，貼上 .torrent 種子檔文件的 Info Hash 特徵碼


上述之磁連產生器裡面的 tracker servers 追蹤者伺服器，最後更新日期為：2019年1月1日

大略流程為，貼上 .torrent 種子檔文件之 info hash 哈希特徵碼，生成一串 Magnet Link 磁力連結，點擊該磁連網址，關聯的BT下載軟體，會自動從 tracker servers 追蹤者伺服器上查找，及獲取對應的 .torrent 種子檔文件，然後打開 .torrent 種子檔文件，開始下載裡面的資源


１、假設您已安裝了 BT 下載軟體 → ( BT 上傳 下載 工具 ) 之使用教學，連結總整理

如果網址後面有帶 info hash 特徵碼，點擊 [按此打開 Magnet Link 磁連網址]

firefox 瀏覽器，就會自動關聯，然後點擊 [開啟連結]




如果網址後面有帶 info hash 特徵碼，點擊 [按此打開 Magnet Link 磁連網址]

chrome 瀏覽器，就會自動關聯，然後點擊 [開啟 BitComet - a BitTorrent Client]




２、假設您從某某處，得到一串 info hash 特徵碼

點擊 [清空] 按鈕，框框內貼上 info hash 特徵碼，

點擊 [按此產成一串 Magnet Link 磁力連結網址] 按鈕，

點擊 [按此打開 Magnet Link 磁連網址] 就會自動關聯，您的 BT 下載軟體




３、假設 BT 下載軟體，沒關聯

什麼是 (剪貼簿) ？

就是 滑鼠右鍵 → 複製 如果是文字/連結/代碼，會暫時放到剪貼簿

點擊 [按此複製磁連網址到剪貼簿]

打開 BT 下載軟體，點擊 [磁連] 按鈕，框框內，滑鼠右鍵 → 貼上




再點擊 [確定] 按鈕，就可以了




４、假設您有比較新的追蹤者伺服器列表，要進行更換

參考 教您幾種簡單的方式，取得 BT 目前可以使用的追蹤者伺服器列表 Ver 3 這篇，可以查找

查看右下角日期，如果太久沒更換，您可以用您自己找的

點擊 [按此清空下方的追蹤者伺服器列表]




ctrl+v貼上，或是滑鼠右鍵 → 貼上

點擊 [按此產成一串 Magnet Link 磁力連結網址] 按鈕，

點擊 [按此打開 Magnet Link 磁連網址] 就會自動關聯，您的 BT 下載軟體




５、分享連結，貼給別人

該網址，尾部會帶一串 info hash 特徵碼，連入這個頁面後，

點擊 [按此打開 Magnet Link 磁連網址] 按鈕，就可自動關聯 BT 下載軟體


教您該如何做，才能獲取 BT 之 .torrent 種子檔內的 info hash 特徵碼

選取任務後，點擊下方的 (任務摘要) 標籤，在 hash 特徵碼上，

滑鼠左鍵壓住，選取反藍後，快鍵 ctrl+c 就可複製




點擊 [清空] 按鈕，快鍵 ctrl+v 貼上，或滑鼠右鍵 → 貼上

點擊 [按此產成一串 Magnet Link 磁力連結網址] 按鈕

點擊分享連結的 [按此複製分享網址到剪貼簿] 按鈕，將你正在下載的 BT 種子之 info hash 特徵碼，黏貼給其他人下載，可以加快下載速度，及延續資源生命
