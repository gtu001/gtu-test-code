啟動mongoDB
    (1)編輯檔案mongod.cfg
          systemLog:
              destination: file
              path: E:\MongoDB\data\log\mongod.log  <--- log路徑 (要把目錄建好:log)
          storage:
              dbPath: E:\MongoDB\data\db <---db目錄 (要把目錄建好:db)
    (2)建立bat檔案
        set mongod="E:\apps\db_tool\mongodb-3.6.2\bin\mongod.exe"  <--設定exe路徑
        set mongo_config="E:\apps\db_tool\mongodb-3.6.2\mongod.cfg"   <--設定cfg路徑
        %mongod%  --config  %mongo_config%  --install  <--啟動
    (3)以系統管理員權限執行bat 
    (4)在cmd輸入 net start MongoDB 按確定

啟動錯誤
  (1) 發生跟服務有關的錯誤: 100.
        進入db資料夾，先刪除 mongod.lock 檔，然後重新開機服務即可；
        要是還不行，就繼續刪 storage.bson文件，然後問題就解決了~
