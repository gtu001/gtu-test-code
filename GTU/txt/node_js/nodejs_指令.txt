初始化專案

	在專案底下
	$ npm init

------------------------------------------------------------------
安裝專案 modules

	在專案底下
	$ npm install 


------------------------------------------------------------------
啟動專案

	在專案底下
	$ npm start [--port 8000]指定port


------------------------------------------------------------------
執行 node.js

	$ node <file_name>.js


------------------------------------------------------------------

安裝 express

	在專案底下
	$ npm install express --save
		ps : --save 會 append 到 package.json

------------------------------------------------------------------

安裝 body-parser
	npm install --save body-parser

	使用方式
	var bodyParser = require('body-parser')
	app.use(bodyParser.json());

------------------------------------------------------------------

安裝 uuid
	npm install --save node-uuid

	使用方式
	var uuid = require('node-uuid');
	let uuidStr = uuid.v4(); //取得uuid

------------------------------------------------------------------

安裝 open api

	npm install --save swagger-tools
