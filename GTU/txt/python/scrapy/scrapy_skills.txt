
安裝 

	---> http://linton.tw/2013/10/18/scapy-network-packet-analysis-tool/
	
	$ sudo apt-get install python-scapy python-pyx python-gnuplot

---------------------------------------------------------------------------

安裝

	$ pip install scrapy

若發生錯誤要安裝 gcc

	$ sudo apt-get install gcc

---------------------------------------------------------------------------
shell執行

	1. 請切到此目錄 /ex2_ptt_beauty/PttBeautyCrawler/>  Ps:有scrapy.cfg那層
	2. scrapy crawl <name>   
		Ps: name 為 spider 內的 name 變數
	3. 非常重要 : allowed_domains 錯了會導致 yield 無效化 （必須無"/",與"https?://")

---------------------------------------------------------------------------

建立project 

	scrapy startproject <project>
