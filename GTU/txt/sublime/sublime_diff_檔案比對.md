# Compare Side-By-Side
---
	ctrl+shift+p  install Package
	輸入 Compare Side-By-Side

操作
## 比對
	先開啟兩個檔案頁簽
	對第一個 頁簽 或 右邊 隨便點右鍵 Compare with ...
	輸入另個檔案名子(局部即可)
	按Enter開啟比對視窗

## 上一段下一段
	Alt+N / Alt+P


## 隨意兩段文字比對
	mark 第一段 --> 選 Mark selection for comparison
	mark 第二段 --> 選 Compare selection
