列出所有user

$ cut -d: -f1 /etc/passwd
$ cat /etc/passwd
$ less /etc/passwd
$ more /etc/passwd


----------------------------------------------------------------------
新增user

$ sudo adduser new_username


----------------------------------------------------------------------
刪除user, home目錄


$ sudo userdel username

$ sudo rm -r /home/username


----------------------------------------------------------------------
修改使用者帳號

$ usermod -l <new_username> <old_username>



----------------------------------------------------------------------
設定user密碼

$ sudo passwd username


----------------------------------------------------------------------
改變user shell

$ sudo chsh username


----------------------------------------------------------------------
改變user 細節

$ sudo chfn username
