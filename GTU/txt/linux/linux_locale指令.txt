查看支援的語系
  $ sudo apt-get install less
  $ less /usr/share/i18n/SUPPORTED
查找特定 
  $ grep -E '[a-z]+_[A-Z]+ UTF-8' /usr/share/i18n/SUPPORTED | grep -E '(tw|TW)'
  $ grep -E '[a-z]+\_[A-Z]+[\s\.\t]UTF\-8' /usr/share/i18n/SUPPORTED  | grep -E '(tw|TW)'
加入台灣語系
  $ sudo echo zh_TW.UTF-8 UTF-8 >> /var/lib/locales/supported.d/local
重建清單
  $ sudo apt-get install debconf
  $ sudo dpkg-reconfigure locales
再跑 (step final) 下載

------
再更新
sudo update-locale LANGUAGE=zh_TW.UTF-8 LC_ALL=zh_TW.UTF-8 LANG=zh_TW.UTF-8
or 
修改 /etc/environment:
加入
LC_ALL=zh_TW.UTF-8
LANG=zh_TW.UTF-8

安裝了那些語系

  $ locale -a
  
-------------------------------------------------------------------------
有哪些可用

  $ locale

-------------------------------------------------------------------------
(step final)建立語系 

  $ sudo locale-gen zh_TW zh_TW.UTF-8 zh_CN.UTF-8 en_US.UTF-8
